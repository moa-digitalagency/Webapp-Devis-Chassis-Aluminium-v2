  🚀 INSTALLATION VPS EN 1 COMMANDE
═══════════════════════════════════════════════════════════

📋 ÉTAPES :

1. Connectez-vous à votre VPS en SSH :
   ssh root@votre-serveur.com

2. Lancez cette commande unique :

   curl -fsSL https://raw.githubusercontent.com/moa-digitalagency/Webapp-Devis-Chassis-Aluminium-v2/main/install-vps.sh | sudo bash

3. Répondez aux 3 questions :
   - Nom de domaine (ex: devis.exemple.com)
   - Email pour SSL
   - Mot de passe base de données

4. Attendez 5 minutes ⏱️

5. C'est prêt ! 🎉
